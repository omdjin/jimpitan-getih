## Jimpitan Getih

Berawal dari [sini](https://twitter.com/Lekday/status/1413119224894017544).

Dibuat dengan [Next.js](https://nextjs.org/).

## Memulai Proyek

Pertama, jalankan:

```bash
yarn
# atau
npm install
```

Kemudian:

```bash
yarn dev
# atau
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di peramban anda.
